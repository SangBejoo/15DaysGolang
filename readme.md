# Golang 15-Day Intensive (with exercises)

A focused 15-day plan to learn Go using exercises from https://dasarpemrogramangolang.novalagung.com/  
Follow one full day per topic, practice with exercises, and track progress.

## How to use
1. Set up Go on your machine.
2. For each day: read the chapter on the linked site, complete exercises, commit code.
3. Track completed exercises in the checklist below.
4. After 15 days, build a small capstone project combining learned topics.
